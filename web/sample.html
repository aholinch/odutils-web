Cart to TLE

Provide a cartesian vector and receive 3 TLEs. The first is generated with an open source SGP4 implementation. The second is USSF SGP4. The third is USSF SGP4-XP.



TLE to Carts

Provide a TLE, start date, end date, and time step. The server will propagate the TLEs (include SGP4-XP) to the specified times and return the cartesian vectors.





SGP4 OD

Provide a set of cartesian vectors and receive SGP4 TLE in response.



SGP4OD Results

Retrieve the OD results.



SGP4-XP OD

Provide a set of cartesian vectors and receive SGP4-XP TLE in response.



SGP4XPOD Results

Retrieve the OD results.
